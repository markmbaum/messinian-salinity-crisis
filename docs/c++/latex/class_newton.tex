\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_newton}{Newton}}\textquotesingle{}s method for nonlinear systems of equations. 

This class implements \mbox{\hyperlink{class_newton}{Newton}}\textquotesingle{}s method for nonlinear systems of equations. The virtual functions F\+\_\+\+Newton and Jac\+\_\+\+Newton allow a derived class to implement the system of equations and its Jacobian matrix.
